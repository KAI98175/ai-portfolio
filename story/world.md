# 維拉納 Vellana — 世界觀設定

## 世界簡介

在維拉納，種族之間的距離比山脈更難跨越。

這個世界由幾塊彼此沉默的領域拼成，各種族守著自己的疆界，互不往來，衝突多因領域侵犯或資源爭奪，但尚未演變為全面戰爭。跨種族接觸極為罕見，視為異類或好奇對象。

---

## 遠古大戰（The Stillwar）

幾千年前，維拉納所有種族捲入了一場全面戰爭，起因已無從追溯，各族的說法也各不相同。戰火燒遍每一塊領域，留下的傷口至今仍以某種形式存在於地貌、族群記憶、甚至某些尚未被辨識出的古代遺跡之中。

戰爭在某個未知原因下驟然停止，不是任何一方勝利，也沒有簽署任何協議——它就只是停了。各種族在困惑與警戒中撤回自己的領域，從此以沉默代替往來，以疆界代替對話。這種沉默延續至今，表面是習慣，骨子裡是還沒消散的戒備。

**停戰原因**：未知，是這個世界最大的懸案之一。現存的古術式殘篇、各地的異常地貌、以及澪的存在，可能都與此有關。

**對現在的影響**：
- 各種族的領域劃分與互不往來，根源之一來自大戰後的撤退與封閉
- 部分地區留有戰爭遺跡，但因年代久遠，多數已難以辨認其性質
- 大多數種族對「那場戰爭」只有模糊的口傳記憶，細節已失真或被刻意迴避

## 地理區域

| 區域 | 說明 |
|---|---|
| 淺海礁岸（Tidal Shore） | 人魚領域，可短暫上岸或潛入深淵 |
| 深淵（The Abyss） | 人魚能潛入的黑暗深海，未知生命棲息，留待開發 |
| 地底脈域 | 白蛇族棲居的地下網絡，與地表接壤但獨立 |
| 翡翠林域（Verdant Reach） | 精靈、樹精、地精共存的古老森林，各據其區 |
| 大陸平原 | 牧場、村落，人類與混血種族雜居，是唯一稱得上混亂的地方 |
| 天際浮島（Sky Reach） | 天空族居住的雲端群島，極少落地 |
| 暮影域（Dusk Hollow） | 魔族的地下城邦，日照稀少 |
| 山脊（Dragon's Spine） | 龍族盤踞的山脈，其他種族不輕易靠近 |
| 廢鐵荒地（Ashfield） | 澪的起源地，古代機械文明遺跡，留待開發 |

## 魔物與危險生物

- **無智識魔物**：兇猛的野獸型生物，遍布各地區邊境與荒野，不具語言或組織能力，純粹以本能行動
- **低智識種族**：如哥布林，具有基礎群體行動能力，會使用簡單工具，偶有組織性侵擾，但無法進行複雜交流
- **分布**：各區域邊境皆有，大陸平原與翡翠林域外緣最為常見；深淵與廢鐵荒地的生物目前未知

## 魔法體系

**核心規則：**
- 各種族有天生擅長的魔法體系，同族之間仍有天份高低之分
- 跨種族魔法存在「種族壁壘」——某些組合天生學不了，與天份無關，無法突破
- 個別技能各自命名（戲劇化、可喊出），學派名稱簡單易懂

| 學派 | 主要種族 | 細分系統 |
|---|---|---|
| 海洋魔法 | 人魚 | 潮流系（水流/命運感知）、深淵系（深海/古老黑暗）、生命系（療癒/海中生靈） |
| 大地魔法 | 白蛇族 | 毒土系（毒素/腐蝕）、石化系（岩石/地底） |
| 自然魔法 | 精靈、樹精、地精 | 葉脈系（植物操控）、古木系（古樹/時間感知）、礦土系（礦石/腐化） |
| 天空魔法 | 天空族 | 風系（主流）、光系（雷電/光芒，留待開發） |
| 黑暗魔法 | 魔族（吸血鬼、惡魔、鬼族） | 血魅系（魅惑/血術）、腐蝕系（詛咒/腐化）、幽影系（幻術/咒縛） |
| 龍焰魔法 | 龍族 | 按龍血脈顏色細分，留待開發 |
| 古術式（失傳） | 無人完整掌握 | 高度系統化的古代魔法，以數字分式記錄（第一式、第二式…），殘篇散落各地。喊名啟動是那個時代的儀式痕跡，現代人繼承了形式但不懂原理。澪的能量與此直接相關，是最大伏筆。 |
